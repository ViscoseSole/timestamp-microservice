Hello. This is a Timestamp Microservice.


How it works: By passing a parameter such as a natural language date or a unix timestamp into the URL we will return an object that has translated one into the other.



Example usage:
https://blooming-tundra-28640.herokuapp.com/January,1,2018
https://blooming-tundra-28640.herokuapp.com/1514757600
Example output:
{ "unix": 1514757600, "natural": "January 1, 2018" }
